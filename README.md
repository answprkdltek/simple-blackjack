## 클래스
게임에 필요한 클래스는 다음과 같다.
* Card: 카드 한 장
* Deck: 카드 뭉치
* Dealer: 딜러
* Player: 참가자
* Hand: 카드 패
* Evaluator: 진행요원
* Game: 게임
